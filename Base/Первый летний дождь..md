# Первый летний дождь.

*Первый летний дождь.
Это слезы ли? Роса?
О, кукушечка!
Мое имя на крыльях
Вознеси за облака!*

Асикага Ёситэру (1536–1565), 13-й сёгун из династии Асикага. Погиб в сражении с вассалами клана Миёси, которые добивались его смещения, чтобы посадить на престол марионеточного сёгуна Асикага Ёсихидэ.
