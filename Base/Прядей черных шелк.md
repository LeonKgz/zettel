# Прядей черных шелк

*Прядей черных шелк
Спутан весь, не разобрать,
Словно этот мир.
Низка яшмовой росы
Вся истает поутру.*

Госпожа Тояма (1553–1582), жена Такэда Кацуёри, мать его сына Нобукацу. Покончила с собой вместе с мужем и детьми.
