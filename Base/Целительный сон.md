# Целительный сон

*Целительный сон
На изголовье из трав
Вернет мои грезы
В далекий
Весенний рассвет.*

Хадзама Мицунобу (1635–1703), один из 47 ронинов. Изголовье из трав — традиционный поэтический образ ночлега в пути. Буддийское мировоззрение представляет жизнь как странствие.
