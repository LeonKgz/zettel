# Нельзя заблудиться, идя

*Нельзя заблудиться, идя
По следу тех, кто ступил
На воинский путь.
Вот почему я не раз
Натягивал лук из катальпы.*

Хорибэ Такэцунэ (1670–1703), один из 47 ронинов.
