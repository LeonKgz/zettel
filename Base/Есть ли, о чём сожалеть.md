# Есть ли, о чём сожалеть

*Есть ли, о чём сожалеть?
Есть ли, что ненавидеть?
Ведь изначально
Самим ходом вещей
Тело идет к покою…*

Суэ Харуката (1521–1555) — полководец в эпоху Сражающихся княжеств, сторонник, затем глава клана Оути. Покончил с собой, проиграв войну с домом Мори.
