# Чем любоваться,

*Чем любоваться,
Пересекая поток,
Невозвратной луной,
Любуйся плывущим облаком,
Из века в век неизменным.*

Танака Кавасаноскэ (1815–1862), роялист из Сацума. Погиб при попытке поднять антисёгунское восстание.
