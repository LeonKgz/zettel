# Tvivler

Сомнение<br><br>В данном случае «сомневающийся» (tvivler) — это не просто человек, не уверенный <br>в себе, раздираемый сомнениями (tvivlende), но скорее «скептик», сознательно выбравший для себя эту позицию.
